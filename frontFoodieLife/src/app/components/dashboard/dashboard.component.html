FoodieLife Demo Page

This page is oriented to show the interface and functionalities of an application to manage the recipes and meals for each day of the week. It's build using Angular, NodeJS and TypeScript.

Update: Ingredients functionality implemented

The ingredients functionality has been added. It lets you add, edit and delete ingredients.
Each ingredient has a price assigned to it that in not yet ready to implement, and it will be used in future functionalities.

Update: Recipes functionality implemented

The recipes functionality has been added. It lets you add, edit and delete recipes.

Update: Ingredients and recipes styling

I added a styling for the app, that easies the interaction with it. Its only thought for Desktop users
It will be changed throughout the development of the app to improve the responsiveness and layout.

Update: Recipe ingredient update

In the recipe edit view has been updated with a set of two tables to choose the desired ingredients and quantity of them for the recipe.
The top table is the one that shows the current recipe's ingredients and the one at the bottom the ones that can be added to the recipe.
It is needed to take in to account that every ingredient that is already on the recipe it will not be shown on the second table.

Update: Ingredients search bar

All the ingredients view now have a search bar that lets look for them by name.
This search bar can be seen at the ingredients view and the recipe edit view.

Update: Ingredients/Recipe price implemented

All the ingredients view now have a price by measure unit. This price can be edited on the "ingredient edit" view.
Now on the all of the recipes view you can check the total price of the ingredients to make it, and on the "detail" and "edit" view you can check the price of each ingredient.
Following that on the "edit view" the total price updates itself every time you delete, add or update an ingredient.

Update: Interface overhaul

The interface has been redesign adding flex to manage the containers, and icons to the majority of the buttons.
